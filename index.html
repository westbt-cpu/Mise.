iOS: Safari → Share → Add to Home Screen

AnyList → mise. (iOS)
iOS doesn't support web app share targets. Use a Shortcut instead — it appears in any app's share sheet:

1. Open Shortcuts → tap +
2. Tap the name at top → rename to mise.
3. Add action → search URL → type your mise URL then add ?cl= at the end
4. Tap the cursor after cl= → tap the token picker → choose Shortcut Input (Shortcuts auto-encodes it)
5. Add action → Open URLs
6. Tap the shortcut name → Add to Share Sheet → set input type to Text

NFC Tags
Location tag URLs
Tap a location to copy its NFC tag URL
iOS setup
1. Write the tag URL to an NFC tag using NFC Tools (free, App Store)
2. In Shortcuts, create a new automation → trigger: NFC → scan your tag
3. Add action: Open URL with the tag URL
4. Add action: Get Clipboard → Open App: Claude (optional — share Claude's response back via copy)
5. Tap the tag → Mise opens to that location's import screen
Android setup
1. Install NFC Tools (free, Play Store)
2. Write a URL record with the tag URL to your NFC tag
3. On Android, tapping the tag opens the URL directly in your default browser — no extra app needed
4. For a photo-to-Claude shortcut use Tasker + HTTP Request or the Claude app's share sheet

Google Sheets setup

Connect mise. to a Google Sheet so your inventory syncs to a spreadsheet you and your household can view from any device.
1
Create a new Google Sheet
Open a blank spreadsheet. The sheet will be populated automatically when you sync.
↗ Open sheets.new
2
Open the Apps Script editor
In your new sheet, click Extensions → Apps Script. Delete any existing code in the editor.
3
Paste the mise. sync script
Copy the code below and paste it into the Apps Script editor.
function doPost(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName('Inventory'); if (!sheet) sheet = ss.insertSheet('Inventory'); sheet.clearContents(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ 'Name','Category','Location', 'Qty','Restock Below','Notes','Expiry','Synced' ]); (data.items || []).forEach(function(item) { sheet.appendRow([ item.name, item.category, item.location, item.qty, item.threshold, item.notes || '', item.expiry || '', new Date().toLocaleString() ]); }); return ContentService .createTextOutput(JSON.stringify({ok:true,count:(data.items||[]).length})) .setMimeType(ContentService.MimeType.JSON); }
4
Deploy as a Web App
In Apps Script, click Deploy → New deployment.
Set Execute as: Me  ·  Who has access: Anyone.
Click Deploy and authorise when prompted.
5
Copy the Web App URL
After deploying, copy the URL shown (starts with https://script.google.com/macros/s/).
6
Paste the URL in Settings
Go back to Settings → Google Sheets Sync, paste the URL in the Script URL field, and tap Save. Then tap Sync ↑ to push your inventory.
